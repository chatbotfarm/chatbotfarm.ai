# ChatbotFarm.ai

Static website for **ChatbotFarm.ai**, hosted on **Cloudflare Pages** and maintained in this GitHub repository.

The site is intentionally framework-free (no build step) and uses shared partials for navigation and footer so global changes are made in one place.

---

## Hosting

- **Platform:** Cloudflare Pages
- **Framework preset:** None
- **Build command:** None
- **Build output directory:** `/` (repo root)

Any push to `main` triggers an automatic redeploy.
